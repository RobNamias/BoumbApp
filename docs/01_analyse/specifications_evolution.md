# Spécifications Fonctionnelles - Évolutions (V2)

Ce document recense les fonctionnalités prévues pour les futures versions de Boumb'App, notamment l'aspect communautaire.

## 1. Aspect Communautaire (Social)
*   **Profil Public :** Page profil enrichie avec Bio, Liens sociaux (SoundCloud, Insta...), et liste des projets publics.
*   **Partage de Projets :**
    *   URL publique unique pour chaque projet.
    *   Player intégré pour écouter sans ouvrir le DAW complet.
    *   Bouton "Fork" / "Remix" pour copier un projet public dans son espace.
*   **Interactions :**
    *   Système de "Likes" / "Favoris".
    *   Commentaires sur les projets.
    *   Système de Tags pour la recherche (ex: "Techno", "Hip-Hop").
*   **Moteur de Recherche :**
    *   Filtrer par Genre, BPM, Date, Popularité.

## 2. Fonctionnalités Avancées DAW
*   **Export Audio :** Rendu du projet en fichier .wav / .mp3.
*   **Automation :** Enregistrement des mouvements de potards en temps réel.
*   **MIDI :** Support des claviers MIDI externes.

## 3. IA Avancée & RAG (Retrieval-Augmented Generation)
*   **Objectif :** Utiliser les données de la communauté pour améliorer l'IA.
*   **Chatbot Assistant :**
    *   Répondre aux questions techniques ("Comment faire une basse Trance ?") en cherchant dans la documentation et les tutoriels indexés.
*   **Génération par Style (Style Matching) :**
    *   Analyser les projets publics (MIDI) pour créer une base de vecteurs.
    *   Permettre à l'utilisateur de demander "Une mélodie comme le projet X" (Recherche de vecteurs similaires).
